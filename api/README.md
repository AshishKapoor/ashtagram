# Getting started

### Create env

`python3 -m venv env`

### Activate env

`source env/bin/activate`

### Install project dependencies

`pip install -r requirements.txt`

### Run backend

`uvicorn main:app --reload`

### Open URL for the swagger ui (API Doc)

`http://localhost:8000/docs`
